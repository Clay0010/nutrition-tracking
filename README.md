 App features
  - AI meal parser (Groq Llama 3.3 70B) — describe meals in plain English
  - Macro tracking with daily targets (calories, protein, carbs, fat)
  - 30-day history with 7-day averages
  - Custom foods library
  - Sign in / sign up auth via Supabase
  - Settings with auto-saved nutrition goals

  Tech stack
  - React 19 + TypeScript + Vite
  - Supabase (Postgres + Auth)
  - Groq API for AI parsing
  - Deployed on Vercel at https://nutrition-tracking-zeta.vercel.app
